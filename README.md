# Análisis Exploratorio del Dataset CTG

Este repositorio contiene un notebook enfocado en explorar el datset **CTG.csv** ver la distribución de las variables, entender sus relaciones y evaluar la calidad del dataset mediante visualizaciones y estadísticas descriptivas.

## ¿Qué puedes encontrar en el notebook?

- Revisión de completitud por columna.
- Identificación de tipos de variables y valores únicos.
- Histogramas con KDE por clase.
- Boxplots para comparar distribuciones entre categorías.
- Barras horizontales para frecuencia de clases.
- Dot plots para comparar valores entre grupos.
- Gráficas de densidad por clase.
- Violin + Swarm plots para ver patrones generales y valores individuales.
- Mapa de correlación entre variables numéricas.
- Interpretaciones breves de cada gráfica.
- Conclusión general basada en el análisis.

## Dataset utilizado

El análisis usa el archivo:

```
CTG.csv
```

Este archivo ya está incluido en el repositorio.

## Cómo correr el notebook

1. Instala **equirements.txt** con:
   ```
   pip install -r requirements.txt
   ```
2. Abre el notebook:
   ```
   Practica3.ipynb
   ```
3. Ejecuta todas las celdas en orden.

Con esto tendrás acceso a todo el análisis y visualizaciones incluidas en el notebook.
